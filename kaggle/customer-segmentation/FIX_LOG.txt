Issue: DataFrame.describe() column selection error

================================================================================
ERROR DETAILS
================================================================================

Location: Feature scaling section
Line: print(df_numerical_scaled.describe()[['mean', 'std']].round(6).T)

Problem:
  - describe() returns a DataFrame with stats as rows
  - Cannot directly select columns like [['mean', 'std']]
  - Need to transpose first: .T, then select columns

Error Message:
  KeyError: "['mean', 'std'] not found in axis"

================================================================================
FIX APPLIED
================================================================================

Changed from:
  print(df_numerical_scaled.describe()[['mean', 'std']].round(6).T)

Changed to:
  desc_stats = df_numerical_scaled.describe().T[['mean', 'std']].round(6)
  print(desc_stats)

Why this works:
  1. describe() creates DataFrame with stats as columns
  2. .T transposes it (stats become rows, features become index)
  3. [['mean', 'std']] now selects the correct columns
  4. .round(6) rounds to 6 decimal places
